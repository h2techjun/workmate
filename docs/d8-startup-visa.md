# D-8 창업·투자 비자 자격 체크리스트

> 외국인이 한국에서 창업/투자해 D-8 비자를 받을 수 있는지 **기본 자격 요건**을
> 체크리스트로 점검. **OASIS 점수는 계산하지 않는다** (출처마다 60~120점으로 제각각·비공개).

## 왜 점수 계산기가 아니라 체크리스트인가

D-8-4(기술창업)의 OASIS 점수제는:

- 공식 채널(HiKorea, oasisvisa.com)은 **최소 60점**(총점 300) 명시
- 일부 로펌/대행 자료는 80점, 120점 이상을 합격선으로 제시 — **불일치**
- 완전한 항목별 배점표는 **법무부 내부 지침으로 비공개**, 연간 개정 가능

→ 잘못된 점수 단정은 창업 비자 준비를 그르칠 수 있으므로, **공식 확인된 고정 요건**만
판정하고 OASIS 점수·배점은 정보로만 안내한다. (F-2-7 도구와 동일 원칙)

## D-8 세부 유형

| 코드 | 명칭 (EN) | 누가 |
|---|---|---|
| D-8-1 | Corporate Investment | 한국 법인을 신설/투자(1억원+, 의결권 10%+)하고 임원으로 활동 |
| D-8-3 | Individual Investment | 한국인이 운영하는 법인에 1억원+ 투자 + 지분 10%+ + 임원 계약 |
| D-8-4 | Technology Startup | 학위 + 지식재산권 보유 외국인이 **신규 법인** 설립, OASIS 점수제 |

(D-8-2 벤처기업은 벤처확인 기업 대상으로 좁아 본 도구에선 가이드 정보로만 안내.)

## 단정 가능한 고정 요건 (법령·공식 근거)

| 항목 | 확정값 | 근거 |
|---|---|---|
| D-8-1/D-8-3 최소 투자금 | 1억원 (₩100,000,000) | 외국인투자촉진법 시행령 §2②  |
| D-8-1/D-8-3 최소 지분 | 의결권 주식 10%+ | 동 시행령 |
| D-8-4 학력 | 국내 전문학사 또는 해외 학사 이상 (취득 완료) | HiKorea 기술창업이민 안내 |
| D-8-4 IP | 특허/실용신안/디자인권 등록 또는 **출원 중** | HiKorea |
| D-8-4 법인 | **신규 법인 설립만** (기존 법인 인수 불가) | HiKorea / OASIS |
| D-8-4 OASIS 필수항목 | 최소 1개 이상 충족 | HiKorea |
| 공통 | 범죄경력 없음 · 출입국 위반 4회 미만 · 세금 미체납 | 공통 비자요건 |
| 신청 전 | 법인등기 + 사업자등록 (투자형은 외국인투자기업 등록) 완료 | OASIS / 출입국 |

## 정보로만 안내 (변동·비공개)

- D-8-4 OASIS 최소 통과 점수 (출처별 60~120점)
- 전체 OASIS 배점표 (법무부 내부 지침, 연간 개정)
- D-8-1 투자금 대비 파견 인원 수 (법령 미명시, 심사관 재량)
- D-8-4 최소 법인 자본금 (법령 명시 없음)
- 사무실 요건 (가상오피스 사실상 불인정 — 실주소 필요, 지소별 상이)

## 체크리스트 로직 (lib/calculations/visa/d8Eligibility.ts)

입력:
- `visaType`: corporateInvestment | individualInvestment | techStartup | none
- 투자형(D-8-1/3): `meetsInvestment`(1억+), `meetsShares`(10%+), `corpRegistered`(법인+사업자/외투등록)
- 기술창업(D-8-4): `hasDegree`, `hasIP`, `newCorp`, `oasisMandatory`(필수항목 1개+)
- 공통: `goodConduct`(범죄경력/출입국/세금)

출력: `baseEligible` / `blockers[]` / `metCount` / `requiredCount` (점수 단정 X)

## 권장 경로

D-10-2 (기술창업준비, 6개월~최대 2년) → 법인 설립 완료 → D-8-4 (1회 12개월) 전환.
