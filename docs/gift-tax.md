# 증여세 계산기 설계 문서

## 개요

한국 증여세(상속세 및 증여세법 제53조·제56조)를 증여재산가액과 증여자 관계로 즉시 산출하는 도구.

## 계산 흐름

```
과세표준 = 증여재산가액 − 증여재산공제 − 10년내 기증여 합산액
산출세액 = 과세표준 × 세율 − 누진공제
납부세액 = 자진신고 시 산출세액 × 0.97 (신고세액공제 3%)
```

## 증여재산공제 (상속세 및 증여세법 제53조, 10년 한도)

| 증여자 관계 | 공제액 |
|---|---|
| 배우자 | 600,000,000원 (6억) |
| 직계존속 → 성년 직계비속 | 50,000,000원 (5천만) |
| 직계존속 → 미성년 직계비속 | 20,000,000원 (2천만) |
| 직계비속 → 직계존속 | 50,000,000원 (5천만) |
| 기타 친족(6촌 이내 혈족·4촌 이내 인척) | 10,000,000원 (1천만) |
| 그 외(타인) | 0원 |

## 누진세율 (상속세 및 증여세법 제56조)

| 과세표준 | 세율 | 누진공제 |
|---|---|---|
| 1억 이하 | 10% | 0 |
| 5억 이하 | 20% | 10,000,000 |
| 10억 이하 | 30% | 60,000,000 |
| 30억 이하 | 40% | 160,000,000 |
| 30억 초과 | 50% | 460,000,000 |

## 입력 명세

| 필드 | 타입 | 설명 |
|---|---|---|
| giftAmount | number (원) | 증여재산가액 |
| donorRelation | enum | 증여자 관계 (6종류) |
| priorGifts | number (원) | 10년내 기증여 합산액 (기본 0) |
| selfReport | boolean | 신고세액공제 3% 적용 여부 |

## 출력 명세

| 필드 | 설명 |
|---|---|
| deduction | 증여재산공제 |
| taxBase | 과세표준 |
| calculatedTax | 산출세액 |
| reportDiscount | 신고세액공제 |
| finalTax | 납부세액 |

## 면책 고지

- 재산평가(시가·보충적 평가) 미반영
- 10년 합산 기증여: 사용자 직접 입력값 사용
- 세대생략할증 30% 미반영
- 가산세(신고불성실·납부지연) 미반영
- 실제 신고는 세무사 또는 국세청 홈택스 이용 권장

## 출처

- 상속세 및 증여세법 제53조 (증여재산공제)
- 상속세 및 증여세법 제56조 (세율)
- 국세청 홈택스 증여세 안내 (https://www.hometax.go.kr)
