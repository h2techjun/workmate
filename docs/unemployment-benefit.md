# 실업급여(구직급여) 계산기 설계 문서

## 개요

고용보험법 제46조·제50조 기반 구직급여(실업급여) 예상 수령액 계산기.
이직 전 평균임금, 나이, 고용보험 가입기간을 입력하면 1일 구직급여액과 총 수령 예상액을 산출한다.

---

## 입력 명세

| 필드 | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `dailyAverageWage` | number (원) | 0 | 이직 전 3개월 임금총액 / 그 기간 총일수 |
| `monthlyAverageSalary` | number (원) | 0 | 월평균임금 (입력 시 /30 환산) — `inputMode: "monthly"` |
| `inputMode` | `"daily" \| "monthly"` | `"daily"` | 평균임금 입력 방식 선택 |
| `ageAbove50` | boolean | false | 만 50세 이상 또는 장애인 여부 |
| `insuranceYears` | number (년) | 1 | 고용보험 가입기간 (년 단위, 1 미만은 1 미만으로 취급) |
| `dailyWorkHours` | number (시간) | 8 | 1일 소정근로시간 (하한 계산용, 기본 8) |

---

## 계산 공식

### 평균임금 결정
- `inputMode === "daily"`: 그대로 사용
- `inputMode === "monthly"`: `dailyAverageWage = monthlyAverageSalary / 30`

### 1일 구직급여액

```
1일 구직급여(raw) = 1일 평균임금 × 60%

상한액 = 66,000원/일  (2019.1.1 이후, 2025 기준)
하한액 = 최저시급 × 0.8 × 1일소정근로시간
       = 10,030원 × 0.8 × 8 = 64,192원  (2025 최저시급 기준)

1일 구직급여 = clamp(raw, 하한, 상한)
             = Math.min(상한, Math.max(하한, raw))
```

### 소정급여일수 (2019.10.1 개정, 이직일 기준)

| 고용보험 가입기간 | 50세 미만 | 50세 이상·장애인 |
|---|---|---|
| 1년 미만 | 120일 | 120일 |
| 1년 이상 ~ 3년 미만 | 150일 | 180일 |
| 3년 이상 ~ 5년 미만 | 180일 | 210일 |
| 5년 이상 ~ 10년 미만 | 210일 | 240일 |
| 10년 이상 | 240일 | 270일 |

### 총 예상 수령액

```
총 수령액 = 1일 구직급여액 × 소정급여일수
```

---

## 출력 명세

| 필드 | 설명 |
|---|---|
| `dailyWageRaw` | 평균임금 × 60% (클램프 전) |
| `dailyWageClamped` | 실제 1일 구직급여액 (상한·하한 클램프 후) |
| `dailyWageFloor` | 하한액 |
| `dailyWageCap` | 상한액 |
| `benefitDays` | 소정급여일수 |
| `totalBenefit` | 총 예상 수령액 |
| `isCapApplied` | 상한 적용 여부 |
| `isFloorApplied` | 하한 적용 여부 |

---

## 법적 근거 및 출처

- 고용보험법 제46조 (구직급여의 수급요건)
- 고용보험법 제50조 (구직급여의 소정급여일수)
- 고용보험법 시행령 제68조 (구직급여 상한·하한)
- 고용24 (work24.go.kr) — 실업급여 안내
- 최저임금위원회 — 2025년 최저임금 10,030원

## 면책 사항

- 실제 수급자격·금액·소정급여일수는 거주지 고용센터가 최종 결정
- 수급요건(이직사유·구직활동·대기기간 등) 별도 충족 필요
- 최저시급은 매년 변동 → 하한액 변동. 2026 최저임금 확정 시 갱신 필요
- 상한액(66,000원)도 정부 고시로 변동 가능
