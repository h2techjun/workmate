# 외국인 건강보험료(NHIS) 추정 계산기 (foreign-health-insurance)

> Korea National Health Insurance (NHIS) premium estimator for foreigners

## 목적
한국에 6개월 이상 체류하는 외국인은 NHIS 의무가입 대상이다. 직장가입자는 보수월액
기반, 지역가입자(프리랜서·무직·유학생)는 외국인 특례로 보험료가 정해진다. "내가 얼마
내는가?"가 가장 큰 혼란이며, 영문으로 계산해주는 도구는 드물다. 본 도구는 가입유형을
고르면 월·연 보험료를 추정한다.

## 핵심 사실 (2026 기준 · 출처 명시)
- **직장가입자**: 건강보험료율 7.19%(노사합산, 근로자 3.595%) + 장기요양보험료
  (건강보험료의 13.14%). 보수월액 보험료 상·하한 적용. (보건복지부 2026 고시)
- **지역가입자 외국인 특례**: `max(소득·재산 점수제 산정액, 전년도 전체 가입자 평균보험료)`.
  대다수 외국인은 평균보험료 하한에 걸려 **약 158,640원/월**(2026, 준확정 — NHIS 매년 11월 고시).
- **유학생(D-2)·초중고(D-4) 50% 경감** → 약 79,320원/월 (연소득 360만 이하 + 재산 1.35억 이하).
- **의무가입**: 입국 6개월 경과 시 지역가입자 당연가입(유학생은 즉시). 미납 시 비자 연장 제한.

## 입력 / 출력
- 입력: `enrollmentType`(employee | regional), `monthlyWage`(직장가입자 보수월액, 원),
  `isStudent`(지역가입자 유학생 50% 경감 여부)
- 출력: 월 보험료(건강 + 장기요양 분리/직장, 평균보험료/지역), 연 환산, 근로자/사용자 부담(직장)

## 공식
```
[직장가입자]
근로자 건강보험료 = clamp(보수월액 × 3.595%, 하한 10,080, 상한 4,591,740)
장기요양보험료   = 근로자 건강보험료 × 13.14%
월 본인부담      = 건강보험료 + 장기요양보험료
(사용자도 동일액 부담 — 참고 표시)

[지역가입자 — 외국인 특례 단순화]
월 보험료 = 평균보험료 158,640원 (일반)
          = 79,320원 (유학생 50% 경감)
```

## 면책 / 한계 (반드시 UI 표기)
1. 지역가입자 소득·재산 점수제 미반영 — 평균보험료 하한으로 단순화. 고소득·고재산 외국인은 더 높을 수 있음.
2. 평균보험료(158,640원)는 매년 11월 NHIS 고시로 변동 — 2026 추정치, 매년 확인 필요.
3. 직장가입자 보수월액 상·하한 캡 적용 시 단순 비례식과 차이.
4. 체류자격·경감 요건은 개인별 소득·재산 심사로 달라짐 — 정확액은 공단(1577-1000) 확인.
5. 참고용 추정. 공식 산정은 NHIS 기준 우선.

## 출처
- 보건복지부 2026년 건강보험료율(7.19%)·장기요양보험료율(13.14%) 결정 보도자료
- 국민건강보험공단 — 재외국민·외국인 지역보험료 부과기준
- 복지부 고시 — 월별 건강보험료액 상·하한
- 대학 국제처 2026 외국인 유학생 건강보험 안내 (50% 경감 79,320원)
