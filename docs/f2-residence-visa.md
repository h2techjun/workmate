# F-2-7 거주(점수제) 비자 자격 체크리스트 (f2-residence-visa)

> Korea F-2-7 Residence Visa — Eligibility Checklist & Guide (NOT a score calculator)

## 목적
외국인이 영주권(F-5) 전 단계로 가장 많이 노리는 F-2-7(점수제 거주) 비자의 **기본
신청 자격 요건**을 체크하고, 점수제(80점)와 절차를 안내한다. 영문 검색 수요가 크다
("F-2 visa korea eligibility/requirements").

## ⚠️ 왜 "점수 계산기"가 아니라 "체크리스트"인가
F-2-7 세부 배점(연령·학력·소득 등)은 **법령이 아니라 법무부 내부 지침(체류자격별
안내매뉴얼, 비공개)**이라 행정사·노무사 출처마다 점수가 다르고, 공식 고정표가 없다.
잘못된 점수 단정은 비자 신청을 그르칠 수 있어 — **점수를 계산해 단정하지 않는다.**
대신 (a) 공식 확인된 기본 자격 요건을 체크하고 (b) 점수 항목은 "정보"로만 안내하며
(c) 정확한 판정은 hikorea/1345로 유도한다.

## 입력 / 출력 (인터랙티브 체크)
- 입력: 신청 유형(5유형 중) · 3년 연속 합법체류 또는 연소득 4천만+ 충족 · 품행단정 ·
  공중보건 요건
- 출력: 기본 신청 자격 충족 여부 + 미충족(blocker) 항목 + "점수 80점 별도 충족 필요" 안내

## 공식 확인된 사실 (체크리스트 근거)
- **신청 5유형**: ①상장법인 종사자 ②유망산업(GNI 1.5배 소득) ③전문직(D-5~E-7) **3년 연속 합법체류**
  ④유학인재(국내 석사+) ⑤잠재 우수인재(이공계 석·박사)
- **3년 체류 면제 조건**: 연소득 4천만원 이상 또는 이공계 유치사업 피초청+추천
- **공통 요건**: 품행단정, 공중보건, 결격사유(금고 이상 형·출입국법 다회위반 등) 없음
- **기준점**: 점수제 총점 **80점 이상**(만점 170 = 공통130+가점40) — 단 세부 배점 비공식
- 취득 후 3년 유지 시 점수제 영주(F-5) 경로

## 면책 / 한계 (반드시 UI 표기)
1. 점수 단정 안 함 — 세부 배점은 법무부 내부 지침으로 비공개·수시 변동, 출처마다 다름.
2. 80점 충족해도 승인 보장 아님 — 신청 유형·체류/소득·품행·공중보건 별도 충족 필요.
3. 개인별 심사관 재량. 정확한 자격은 hikorea.go.kr / 이민정보센터 1345 확인.
4. 신청 유형·소득 기준(GNI 연동)은 매년 변동. 2026-06 기준 조사.
