# 중도상환수수료 계산기 설계 문서

> Claude Code는 이 문서를 기반으로 정확한 계산 로직을 구현해야 합니다.

## 📚 참고 출처

- **은행 여신거래기본약관** — 중도상환수수료 슬라이딩 방식
- **각 은행 여신상품설명서** (국민은행·우리은행·하나은행·신한은행 등 공통 약관)
- **금융감독원 금융소비자정보포털** — 중도상환수수료 안내

## 계산 공식 (슬라이딩 방식)

```
중도상환수수료 = 중도상환원금 × 수수료율 × (잔존일수 / 면제기준기간일수)
```

### 변수 정의

| 변수 | 설명 | 기본값 |
|---|---|---|
| 중도상환원금 (원) | 대출 만기 전 일시 또는 부분 상환하는 금액 | 입력 |
| 수수료율 (%) | 은행·상품별 약정 수수료율 | 1.2% |
| 대출 경과 기간 (개월) | 대출 실행일로부터 중도상환 예정일까지 | 입력 |
| 면제기준기간 (년) | 이 기간 경과 후 수수료 면제 | 3년 |

### 일수 계산

```
면제기준기간일수 = 면제기준기간(년) × 365
경과일수 = 경과개월 × 30.4  (단순화 가정: 1개월 = 30.4일)
잔존일수 = max(0, 면제기준기간일수 − 경과일수)
```

- 잔존일수가 0 이하이면 수수료 면제 (경과 기간 ≥ 면제기준기간).
- 실효 수수료율 = 수수료액 / 중도상환원금 × 100

### 수치 예시

| 경과 | 잔존일수 (3년 기준) | 수수료율 1.2%, 1억 원 |
|---|---|---|
| 0개월 | 1,095일 | 1,314,000원 |
| 12개월 | 730일 | 876,000원 |
| 24개월 | 365일 | 438,000원 |
| 36개월 이상 | 0일 | 면제 |

### 가정 및 한계

1. **1개월 = 30.4일** 단순화 (실제 은행은 대출실행일~상환일 달력 일수 계산).
2. **수수료율·면제기간**: 은행·상품·실행시점별 상이. 고정금리/변동금리 다름.
3. **일부 상품**: 3년 이전 조기상환 수수료 없는 경우 있음 (인터넷전문은행 등).
4. **원금 균등 차감**: 잔여 대출 원금 기준으로 계산하는 방식 미반영 (본 계산기는 입력 원금 기준).
