# 산업용 전기 계산기 설계 문서

> Claude Code는 이 문서를 기반으로 정확한 계산 로직을 구현해야 합니다.
> 모든 공식과 상수의 출처는 명시되어야 하며, KS 표준을 우선합니다.

## 📚 참고 표준

- **KS C IEC 60364** (저압 전기설비)
- **KS C IEC 60364-5-52** (배선설비 - 케이블 시스템)
- **한국전기설비규정 (KEC)** 2021 개정판
- **내선규정** (한국전기기술인협회)

## 1. 전선 굵기 계산기 (Wire Size Calculator)

### 입력
| 항목 | 단위 | 설명 |
|---|---|---|
| 전압 (V) | V | 정격 전압 (220, 380, 440 등) |
| 전류 (I) | A | 부하 전류 |
| 거리 (L) | m | 전원에서 부하까지 편도 거리 |
| 상 종류 | - | 단상 2선식 / 단상 3선식 / 3상 3선식 / 3상 4선식 |
| 도체 재질 | - | 동(Copper) / 알루미늄(Aluminum) |
| 허용 전압강하율 | % | 일반적으로 2~5% (내선규정 기준) |
| 배선 방식 | - | 직접매설 / 전선관 / 케이블 트레이 등 |
| 주위 온도 | °C | 기본값 30°C |

### 출력
| 항목 | 단위 | 설명 |
|---|---|---|
| 권장 단면적 | mm² | KS 규격 단면적 (1.5, 2.5, 4, 6, 10, 16, 25, 35, 50, 70, 95, 120, 150, 185, 240, 300, 400) |
| 실제 전압강하 | V | 계산된 전압강하 값 |
| 전압강하율 | % | 실제 강하 / 정격 전압 × 100 |
| 허용 전류 | A | 선정된 전선의 허용 전류 |
| 안전 여유율 | % | (허용전류 - 부하전류) / 허용전류 × 100 |

### 핵심 공식

#### 전압강하 (내선규정)

**단상 2선식**:
```
e = (35.6 × L × I) / (1000 × A)
```

**3상 3선식**:
```
e = (30.8 × L × I) / (1000 × A)
```

**단상 3선식 / 3상 4선식**:
```
e = (17.8 × L × I) / (1000 × A)
```

여기서:
- `e`: 전압강하 (V)
- `L`: 거리 (m)
- `I`: 전류 (A)
- `A`: 전선 단면적 (mm²)

> 위 계수(35.6, 30.8, 17.8)는 동선 기준. 알루미늄은 약 1.6배 적용.

#### 전선 단면적 역산

허용 전압강하 `e_allow`로부터 최소 단면적:
```
A_min = (계수 × L × I) / (1000 × e_allow)
```

이후 KS 규격 단면적 중 `A_min` 이상의 가장 작은 값을 선정.

### KS 규격 단면적 (mm²)
```
1.5, 2.5, 4, 6, 10, 16, 25, 35, 50, 70, 95, 120, 150, 185, 240, 300, 400, 500, 630
```

### 허용 전류 (KS C IEC 60364-5-52 부속서 B 기준)

**A1 방식** (단열벽 내 전선관, 동선, PVC 절연, 30°C):
| 단면적 (mm²) | 단상 (A) | 3상 (A) |
|---|---|---|
| 1.5 | 17.5 | 15.5 |
| 2.5 | 24 | 21 |
| 4 | 32 | 28 |
| 6 | 41 | 36 |
| 10 | 57 | 50 |
| 16 | 76 | 68 |
| 25 | 101 | 89 |
| 35 | 125 | 110 |
| 50 | 151 | 134 |
| 70 | 192 | 171 |
| 95 | 232 | 207 |
| 120 | 269 | 239 |
| 150 | 309 | 275 |
| 185 | 353 | 314 |
| 240 | 415 | 369 |
| 300 | 477 | 424 |

> **주의**: 배선 방식, 주위 온도, 동시 사용 회로 수에 따라 보정 계수 적용 필요.
> MVP에서는 A1 방식 + 30°C 기준만 우선 구현, 추후 확장.

## 2. 차단기 용량 계산기 (Breaker Capacity Calculator)

### 입력
| 항목 | 단위 | 설명 |
|---|---|---|
| 부하 전류 | A | 정격 부하 전류 |
| 부하 종류 | - | 일반 부하 / 모터 부하 / 인버터 부하 |
| 안전율 | - | 1.25 (일반) ~ 2.0 (모터 기동) |

### 출력
| 항목 | 단위 | 설명 |
|---|---|---|
| 권장 차단기 정격 | A | KS 표준 정격 중 선정 |
| 차단기 종류 | - | MCCB / ELB / MCB |
| 안전 여유율 | % | - |
| 권장 사용 케이블 | mm² | 차단기 용량에 맞는 전선 굵기 |

### 핵심 공식

```
정격_차단기 = ceil(부하전류 × 안전율, 표준_정격_리스트)
```

### KS 표준 차단기 정격 (A)
```
MCB:   6, 10, 16, 20, 25, 32, 40, 50, 63
MCCB:  15, 20, 30, 40, 50, 60, 75, 100, 125, 150, 175, 200, 225, 250, 300, 350, 400, 500, 600, 800, 1000, 1200, 1600, 2000
ELB:   동일 (감도 30mA / 100mA / 300mA 옵션)
```

### 부하별 안전율 가이드
| 부하 종류 | 안전율 | 비고 |
|---|---|---|
| 일반 조명/콘센트 | 1.25 | 내선규정 기본 |
| 일반 모터 (직입기동) | 1.25 ~ 1.5 | 기동전류 고려 |
| 인덕션 모터 (Y-Δ 기동) | 1.5 ~ 2.0 | - |
| 인버터 부하 | 1.1 ~ 1.25 | 소프트스타트 |
| 용접기 | 2.0 ~ 3.0 | 단속 부하 |

### ⚠️ AHES 실무 노하우 (마스터 전용)
- **인버터 부하 + ELB**: Y-캐패시터 누설전류로 30mA ELB 트립 빈번
- **권장**: 인버터 회로는 ELB 대신 **MCB/MCCB 사용**
- **마스터 경험**: HSE-62b → HSB-102b 교체 사례 활용 가능

## 3. 전압강하 계산기 (Voltage Drop Calculator)

### 입력
| 항목 | 단위 | 설명 |
|---|---|---|
| 전압 | V | 송전단 전압 |
| 전류 | A | 부하 전류 |
| 거리 | m | 편도 거리 |
| 단면적 | mm² | 전선 단면적 |
| 상 종류 | - | 단상 2선식 / 3상 3선식 / 단상 3선식 / 3상 4선식 |
| 도체 재질 | - | 동 / 알루미늄 |

### 출력
| 항목 | 단위 | 설명 |
|---|---|---|
| 전압강하 | V | 절대값 |
| 전압강하율 | % | (강하/정격) × 100 |
| 수전단 전압 | V | 정격 - 강하 |
| 적합 여부 | - | 내선규정 허용치 대비 OK/NG |

### 내선규정 허용 전압강하

| 길이 | 허용 강하율 |
|---|---|
| 60m 이하 | 3% |
| 120m 이하 | 5% |
| 200m 이하 | 6% |
| 200m 초과 | 7% |

> 단, 수용가 내 인입 후는 일반적으로 **2% 이내** 권장.

### 공식
위 1번 항목 "전압강하" 공식과 동일.

## 🧪 테스트 케이스 (필수)

각 계산기는 다음 케이스로 검증해야 합니다.

### 전선 굵기 - Test 1
- 입력: 380V 3상, 100A, 50m, 2% 허용
- 기대 단면적: 35mm² (동선)
- 검증: `(30.8 × 50 × 100) / (1000 × A) ≤ 7.6V` → A ≥ 20.26 → 25mm² 또는 35mm²

### 차단기 용량 - Test 1
- 입력: 부하 80A, 일반 모터, 안전율 1.25
- 계산: 80 × 1.25 = 100A
- 기대: MCCB 100A

### 전압강하 - Test 1
- 입력: 380V 3상, 50A, 100m, 16mm²
- 계산: (30.8 × 100 × 50) / (1000 × 16) = 9.625V
- 강하율: 9.625 / 380 × 100 = 2.53%
- 기대: 강하 9.6V, 강하율 2.53%

## 🎨 UI/UX 가이드

### 공통 요구사항
- **모바일 우선** (현장에서 폰으로 사용)
- 입력값 실시간 검증 (Zod)
- 계산 과정을 **단계별로 표시** (교육 효과 + SEO)
- 결과를 **PDF/이미지로 다운로드** 가능
- 공식과 KS 출처를 항상 페이지 하단에 명시

### 단계별 표시 예시
```
[전선 굵기 계산 결과]
✅ 권장 단면적: 35mm² (KS C IEC 60227-3)
✅ 안전 여유율: 24%

[계산 과정]
1단계: 허용 전압강하 산정
  380V × 2% = 7.6V

2단계: 최소 단면적 계산
  A_min = (30.8 × 50 × 100) / (1000 × 7.6)
       = 20.26mm²

3단계: KS 표준 단면적 선정
  20.26mm² 이상 → 25mm² (다음 표준치)
  안전 여유 고려 → 35mm² 권장

[출처]
- 내선규정 1410-1 (전압강하)
- KS C IEC 60364-5-52 (허용전류)
```
