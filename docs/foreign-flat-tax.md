# 외국인 단일세율 vs 누진세 비교 계산기 (foreign-flat-tax)

> Korea Foreign Worker Flat Tax (19%) vs Progressive Tax — which saves more?

## 목적
한국에서 일하는 외국인 근로자는 **19% 단일세율 특례**(조세특례제한법 §18조의2)와
**일반 누진세율**(6~45%) 중 유리한 쪽을 선택할 수 있다. 어느 쪽이 유리한지는
연봉·공제 규모에 따라 갈리며, 기존 영문 자료는 글로만 설명한다. 본 도구는 연봉을
입력하면 두 방식의 세액을 즉시 계산·비교한다.

## 핵심 사실 (2026 귀속 기준 · 출처 명시)
- **단일세율**: 소득세 19% + 지방소득세 1.9%(소득세의 10%) = **합산 20.9%**
  (조세특례제한법 §18조의2). 모든 비과세·공제·감면 배제.
- **신규 진입 일몰**: 2026-12-31까지 국내 최초 근로 시작분만 적용 가능.
  적용 시 최초 근로일부터 **20년**간 유지.
- **누진세**: 소득세법 §55 8구간(6~45%) + 지방소득세 10% 별도. `incomeTax.ts`의
  `BRACKETS_2026` 재사용.
- **근로소득공제**(§47), **인적공제**(본인+부양가족 1인당 150만), **근로소득세액공제**(§59) 반영.
- **손익분기**: 통념(8천만)과 달리 실제로는 **약 1.3~1.7억**(공제 많을수록 상승).
  단일세율은 고소득·소수공제자에게만 유리.

## 입력 / 출력
- 입력: `grossSalary`(연 총급여, 원), `dependents`(부양가족 수, 본인 제외, 기본 0)
- 출력: 단일세율 세액(소득세+지방세+합산), 누진세 세액(과세표준·산출·세액공제·결정+지방세),
  유리한 쪽, 절세액 차이, 양쪽 실효세율

## 공식
```
[단일세율]
flatIncomeTax = grossSalary × 0.19
flatLocalTax  = flatIncomeTax × 0.10
flatTotal     = flatIncomeTax + flatLocalTax           // = grossSalary × 0.209

[누진세] (거주자 기준)
근로소득공제   = §47 구간식 (한도 2,000만)
인적공제       = 1,500,000 × (1 + dependents)
과세표준       = max(grossSalary − 근로소득공제 − 인적공제, 0)
산출세액       = 과세표준 × 세율 − 누진공제액           // BRACKETS_2026
근로소득세액공제 = §59 (산출세액 130만 이하 55%, 초과 30%, 총급여별 한도)
결정세액       = max(산출세액 − 근로소득세액공제, 0)
progLocalTax   = 결정세액 × 0.10
progTotal      = 결정세액 + progLocalTax

[비교]
유리한쪽 = flatTotal < progTotal ? "flat" : "progressive"
```

## 면책 / 한계 (반드시 UI 표기)
1. 4대보험(국민연금·건강·고용·산재) 미반영 — 실수령액 아님, 소득세+지방세만.
2. 특별세액공제(의료비·교육비·기부금·연금저축 등)·신용카드 공제 미반영 →
   공제 많을수록 누진세가 더 유리해져 손익분기 상승.
3. 단일세율 측 회사부담 보험료 과세산입·비과세 급여 과세 미반영 → 단일세율 실부담이 클 수 있음.
4. 거주자 기준. 비거주자는 공제 제한으로 결과가 다름.
5. 단일세율 신규 진입 일몰 2026-12-31(연장 입법 시 변동).
6. 참고용 추정 — 정확한 신고는 국세청 홈택스/세무 전문가 확인 필요.

## 출처
- 조세특례제한법 §18조의2 (단일세율 특례·일몰·20년)
- 소득세법 §47(근로소득공제) · §55(세율) · §59(근로소득세액공제)
- 국세청 NTS 종합소득세 세율표 (2026 귀속)
