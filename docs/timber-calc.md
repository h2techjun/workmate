## 1. 서까래 계산기 (Rafter Calculator)

서까래(rafter)는 지붕의 경사를 따라 마룻대(ridge)부터 처마(eave)까지 이어지는 구조 부재.

### 입력
| 항목 | 단위 | 설명 |
|---|---|---|
| 런 (Run) | mm | 마룻대에서 외벽까지의 수평 거리 (1/2 건물 폭) |
| 라이즈 (Rise) | mm | 외벽 상단부터 마룻대까지의 수직 높이 |
| 처마 길이 | mm | 외벽에서 서까래 끝까지의 수평 길이 (선택) |

### 출력
| 항목 | 단위 | 설명 |
|---|---|---|
| 기본 서까래 길이 | mm | 외벽~마룻대까지의 빗변 길이 |
| 처마 추가 길이 | mm | 처마 부분 빗변 길이 |
| 전체 서까래 길이 | mm | 절단해야 할 총 길이 |
| 경사 각도 | ° | 수평선 대비 각도 |
| 구배율 | % | (rise / run) × 100 |
| 피치 비율 | rise/12 | 미국식 표기 (예: 4/12) |
| 면적 보정 계수 | - | sec(θ) — 평면적 → 실제 지붕면적 환산 |

### 핵심 공식
```
기본 길이 = √(run² + rise²)
처마 길이 = eave × sec(θ) = eave × √(1 + (rise/run)²)
각도 = atan(rise / run)
구배율 = (rise / run) × 100
면적 보정 = 1 / cos(θ) = sec(θ)
```

### 예제 검증
- run=4000, rise=1000, eave=600
- baseLength = √(16,000,000 + 1,000,000) = √17,000,000 ≈ 4123.1 mm
- 각도 = atan(1000/4000) = atan(0.25) ≈ 14.04°
- 구배율 = 25%
- 피치 비율 = 3.0/12

## 2. 지붕 경사 계산기 (Roof Pitch Calculator)

세 가지 표현(각도, 구배율%, 피치 rise/12) 사이의 양방향 변환.

### 입력 (셋 중 하나)
| 모드 | 입력 | 설명 |
|---|---|---|
| `angle` | 각도(°) | 0 < angle < 90 |
| `slope` | 구배율(%) | 0 < slope (0=평지붕) |
| `ratio` | rise:12 | rise만 입력 (run=12 고정) |

### 출력
- 세 가지 표현 모두 동시 출력
- 분류 (평지붕 / 저경사 / 중경사 / 급경사 / 초급경사)
- 면적 보정 계수 sec(θ)
- 적용 가능 마감재 추천 (참고용)

### 분류 (한국건축 관행)
| 각도 | 분류 |
|---|---|
| ≤ 10° | 평지붕 (방수 마감 권장) |
| ≤ 30° | 저경사 (아스팔트 슁글, 금속) |
| ≤ 45° | 중경사 (점토기와, 슁글, 금속) |
| ≤ 60° | 급경사 (점토기와, 천연석재) |
| > 60° | 초급경사 (장식적 요소) |

## 3. 계단 계산기 (Stair Calculator)

총 높이와 권장 단높이로부터 안전한 단수·단너비 산정. **건축법·Blondel 공식**으로 검증.

### 입력
| 항목 | 단위 | 설명 |
|---|---|---|
| 총 높이 (Total Rise) | mm | 1층 마감면 → 2층 마감면 |
| 총 진행 거리 | mm | 수평 진행 거리 (선택) |
| 권장 단높이 | mm | 목표 라이저 (예: 180mm) |
| 용도 | - | 주거용 / 공용 |

### 출력
| 항목 | 단위 | 설명 |
|---|---|---|
| 단 개수 | 개 | 라이저 수 |
| 실제 단높이 | mm | totalRise / 단수 |
| 단너비 | mm | totalRun / (단수-1) 또는 Blondel 추천 |
| 계단 길이 (빗변) | mm | √(totalRise² + 진행거리²) |
| 계단 각도 | ° | atan(riser / tread) |
| Blondel 값 | mm | 2R + T — 600~640 사이 권장 |

### 안전 기준 (건축법 시행령 제48조 + KEC)
| 용도 | 최소 단너비 | 최대 단높이 | 최소 폭 | Blondel |
|---|---|---|---|---|
| 주거용 (단독·세대내) | 240mm | 200mm | 750mm | 600~640mm |
| 공용 (공공·공동주택공용부) | 280mm | 180mm | 1200mm | 600~640mm |

### 핵심 공식
```
단수 = round(총높이 / 권장단높이)
실제단높이 = 총높이 / 단수
단너비 (입력 시) = 진행거리 / (단수 - 1)
단너비 (Blondel) = 620 - 2 × 실제단높이
빗변 = √(총높이² + (단너비 × (단수-1))²)
각도 = atan(실제단높이 / 단너비)
Blondel = 2 × 실제단높이 + 단너비
```

### 경고
- 실제단높이 > 최대단높이: 단수 증가 권장
- 단너비 < 최소단너비: 진행거리 증가 권장
- Blondel < 600 또는 > 640: 보행 불편

## 4. 목재 수량 계산기 (Lumber Quantity)

목재 부재의 부피·보드피트·재(才)·무게 환산.

### 입력
| 항목 | 단위 | 설명 |
|---|---|---|
| 폭 | mm | 단면 폭 |
| 두께(또는 높이) | mm | 단면 두께 |
| 길이 | mm | 부재 길이 |
| 수량 | 개 | 부재 개수 |
| 수종 | - | 소나무/낙엽송/삼나무/SPF 등 |

### 출력
| 항목 | 단위 | 설명 |
|---|---|---|
| 총 부피 | m³ | (W × H × L × Qty) / 10⁹ |
| 보드피트 | BF | 미국식 단위 |
| 재(才) | 才 | 일본·한국 전통 단위 (1才 ≈ 0.00334 m³) |
| 추정 무게 | kg | 부피 × 비중 |

### 핵심 공식
```
부피(m³) = (W × H × L × Qty) / 1,000,000,000
보드피트(BF) = 부피(in³) / 144
            = (W_in × H_in × L_in × Qty) / 144
재(才) = 부피(m³) / 0.003342  (1才 = 1치×1치×12자)
무게(kg) = 부피(m³) × 비중(kg/m³)
```

### 수종별 비중 (kg/m³, 기건 상태)
| 수종 | 비중 |
|---|---|
| 소나무 (Pine) | 500 |
| 낙엽송 (Larch) | 540 |
| 삼나무 (Cedar) | 380 |
| SPF (가문비/소나무/전나무) | 460 |
| 더글러스 퍼 (Douglas Fir) | 530 |
| 참나무 (Oak) | 750 |
